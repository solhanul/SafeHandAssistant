# 손안의 안심비서

고령층이 스마트폰을 쉽고 안전하게 사용하도록 돕는 Kotlin·Jetpack Compose 기반 Android MVP입니다.

## 현재 구현된 화면 흐름

`첫 안내 → 홈 → 위험 확인 / 안심비서 설정`

- 큰 글씨·높은 대비·큰 터치 영역의 홈 화면
- 첫 실행 안내 화면
- 화면 위 FAB를 통해 현재 화면의 글자를 TTS로 읽어 주기
- URL의 HTTPS 여부, 단축 URL, 피싱 유도 단어, 복잡도를 이용한 규칙 기반 위험도 안내와 진동 경고
- 보호자 번호로 전화 앱 또는 문자 앱 열기
- 접근성 서비스를 켜면 다른 앱 위에도 `듣기`·`검사` 빠른 도움 버튼 표시
- `듣기`는 화면 캡처를 기기 안에서 OCR 처리해 실제 보이는 글자만 읽음 (Android 11 이상)
- 메시지·카카오톡·브라우저 등 화면에서 감지한 의심 링크는 진동·음성으로 경고
- 설정에서 TTS 음량·속도 조절, 튜토리얼 다시보기, 화면 위 빠른 도움(FAB) 표시/숨김 설정
- 첫 실행 및 설정의 ‘튜토리얼 다시 보기’에서 홈 화면·FAB·글자 읽기·링크 검사 사용 순서 안내

접근성 서비스는 비밀번호 입력란을 읽지 않으며, 화면 캡처는 OCR 처리 직후 메모리에서 폐기됩니다. 읽은 문장·주소·화면은 저장하거나 외부로 전송하지 않습니다.

## Android Studio에서 실행하기

1. Android Studio에서 이 폴더를 엽니다.
2. Android SDK Platform 35를 설치합니다.
3. SDK 위치를 선택한 뒤 동기화하고 에뮬레이터 또는 기기에서 실행합니다.
4. 앱 홈의 **화면 위 빠른 도움 켜기**를 누르고, 시스템 설정에서 `손안의 안심비서 빠른 도움` 접근성 서비스를 켭니다.

이 프로젝트는 한글 경로에서도 빌드되도록 `gradle.properties`에 경로 호환 설정을 포함합니다.
