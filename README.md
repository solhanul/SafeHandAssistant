# 손안의 안심비서

고령층이 스마트폰을 쉽고 안전하게 사용하도록 돕는 Kotlin·Jetpack Compose 기반 Android MVP입니다.

## 현재 구현된 화면 흐름

`첫 안내 → 홈 → 글자 읽기 / 위험 확인 / 가족 도움 요청`

- 큰 글씨·높은 대비·큰 터치 영역의 홈 화면
- 첫 실행 안내 화면
- 입력한 문장을 Android TTS로 읽어 주기
- URL의 HTTPS 여부, 단축 URL, 피싱 유도 단어, 복잡도를 이용한 규칙 기반 위험도 안내와 진동 경고
- 보호자 번호로 전화 앱 또는 문자 앱 열기
- 긴급전화(112) 연결

## Android Studio에서 실행하기

1. Android Studio에서 이 폴더를 엽니다.
2. Android SDK Platform 35를 설치합니다.
3. SDK 위치를 선택한 뒤 동기화하고 에뮬레이터 또는 기기에서 실행합니다.

이 프로젝트는 한글 경로에서도 빌드되도록 `gradle.properties`에 경로 호환 설정을 포함합니다.
